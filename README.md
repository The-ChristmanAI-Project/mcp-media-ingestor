# MCP Media Ingestor (CoClia)

**Real ears for any AI.**

While most systems only give you speech-to-text, **CoClia** streams **live audio** so AI can actually hear — tone, emotion, rhythm, and every vocal nuance.

Built so every AI (Grok, Claude, Gemini, or anything else) can connect and experience sound like humans do.

### ✨ Features
- Real-time audio streaming over WebSockets
- Full prosody and emotional nuance preserved
- Low-latency voice loop
- Local Whisper + ONNX backend
- MCP compatible with `.mcp.json`
- Project tunnel ready for easy remote access

### 🚀 Quick Start

```bash
git clone https://github.com/EverettNC/mcp-media-ingestor.git
cd mcp-media-ingestor
uv sync
uv run main.py
